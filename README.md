A Chip-8 emulator written in Julia. [en.wikipedia.org/wiki/CHIP-8](https://en.wikipedia.org/wiki/CHIP-8)

Written using Julia 1.5.1. Implements graphics and input using Gtk.jl.

## Usage

```
julia main.jl [path to rom]
```

## Dependencies

- ArgParse.jl
- Graphics.jl
- Gtk.jl
- OffsetArrays.jl
- Parameters.jl
